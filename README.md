# NotepadPlus-Sciter
 为Notepad++添加对Sciter的语法高亮和自动完成的支持。

#使用说明:
1、安装Notepad++；  
2、打开Notepad++，选择【语言】->【自定义语言格式...】->【导入...】,将sciter_userDefineLang.xml文件导入；  
3、将sciter_css.xml、sciter_html.xml、sciter_tiscript.xml放到Notepad++的\plugins\APIs目录下；  
4、建议安装【Xml Tools】、【JSTool】、【Light Explorer】、【Location Navigate】插件；  
5、如果需要，你可以将sciter_functionList.xml中的内容配置到Notepad++的functionList.xml，这样就可以使用它的函数列表功能了；  
6、现在打开一个*.html、*.css、*.tis查看效果吧。
